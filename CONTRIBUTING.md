# Contributing to Agent Manifest Protocol

欢迎！AMP 是一个开放社区协议，任何形式的贡献都欢迎。

## 贡献方式

### 📖 完善规范
协议规范是 `spec/manifest.md`。如果你发现遗漏、矛盾或改进空间，欢迎提 PR。

### 🐛 报告 Bug
使用 Bug Report 模板提 Issue。

### 💡 提功能建议
在 Discussions 或 Feature Request 模板中提出。

### 📦 注册你的 Skill
将 `agent.json` 提 PR 到 `registry/` 目录。

### 🌐 实现 Runtime 兼容
在你自己构建的 AI Runtime 中实现 AMP 协议解析。

## PR 流程

1. Fork 本仓库
2. 创建你的分支 (`git checkout -b feat/my-feature`)
3. 提交修改 (`git commit -m 'feat: add ...'`)
4. 推送到你的分支 (`git push origin feat/my-feature`)
5. 提 Pull Request

## 代码规范

- CLI 代码：ESM (Node.js), 遵循现有风格
- 协议规范：Markdown + JSON Schema
- 搜索引擎：Node.js

## 行为准则

请保持友善和建设性。技术分歧很正常，人身攻击不接受。
